# PhoneAppAPI

Backend system for managing phone brands, models, features, sales, and inventory.  
Built with **ASP.NET Core** and **Entity Framework Core**.

## Features
- Manage brands and models
- Add and update features
- Track sales and restock movements
- Maintain price history
- Soft delete functionality

## Technologies
- ASP.NET Core
- Entity Framework Core
- MySQL
- Swagger for API documentation

## How to Run
1. Clone the repository
2. Run `dotnet run`
3. Open Swagger at `https://localhost:5001/swagger`
